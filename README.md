## Getting Started

### Installation
First make sure you have [Rojo](https://marketplace.visualstudio.com/items?itemName=evaera.vscode-rojo) 
    & [Roblox Studio](https://www.roblox.com/create) installed and set up.

Next, install Luau Language Server for Visual Studio Code:
    [Luau Language Server](https://marketplace.visualstudio.com/items?itemName=JohnnyMorganz.luau-lsp)

### Building and Running
To build the place from scratch, use:

```bash
./scripts/rojo.exe build -o "fullstack_project.rbxlx"
```

Next, open `fullstack_project.rbxlx` in Roblox Studio and start the Rojo server:

```bash
./scripts/rojo.exe serve
```

## Deployment

To deploy your place to Roblox, you can use the `rojo upload` command.

```bash
./scripts/rojo.exe upload --asset_id [PLACE ID] --cookie "[SECURITY COOKIE]"
```

If you are on Windows and have Roblox Studio installed, --cookie is optional and will be pulled from your Roblox Studio session if not specified.

For more help, check out [the Rojo documentation](https://rojo.space/docs).
